# Sports Team Performance Analytics Bot

Telegram bot for sports coaches and athletic directors tracking team performance metrics and training load. Coaches submit match results and player ratings after games, bot calculates rolling form scores, injury risk flags based on training load accumulation, player availability tracker with return-to-play timelines, weekly performance digest, and season statistics dashboard. Players receive individual performance breakdowns and next session training targets. Supports multiple sports including football, basketball, tennis, and cricket with sport-specific performance metrics. Multi-team support for academies and clubs.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
